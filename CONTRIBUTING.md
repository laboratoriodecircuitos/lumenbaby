# Contribuição e Uso do Repositório

Este repositório faz parte do desenvolvimento público do Projeto LumenBABY, primeiro produto da marca LumenSense.

No momento, o projeto é conduzido por Rafael Albuquerque / Laboratório de Circuitos.

## Diretrizes

- Manter o projeto simples, replicável e didático.
- Priorizar Arduino.
- Não propor troca para ESP32, Raspberry Pi Pico ou outra plataforma nesta fase.
- Não adicionar Wi-Fi, Bluetooth, aplicativo ou recursos fora do escopo inicial.
- Não inserir dados sensíveis.
- Não inserir arquivos privados.
- Manter documentação atualizada quando houver mudanças relevantes.

## Filosofia técnica

A LumenBABY prioriza:

- simplicidade;
- replicabilidade;
- custo acessível;
- experiência visual;
- autonomia suficiente;
- estabilidade prática;
- valor didático.

O objetivo não é criar uma solução industrial de ultra eficiência, mas sim uma luminária inteligente funcional, bonita, reproduzível e adequada como estudo de caso real.
