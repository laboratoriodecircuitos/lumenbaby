# 01 — TAP Resumido do Projeto LumenBABY

## Nome do projeto

Desenvolvimento da Luminária Nuvem Inteligente LumenBABY.

## Marca

LumenSense.

## Produto

LumenBABY.

## Objetivo geral

Desenvolver, testar e documentar uma luminária decorativa infantil inteligente, com bateria interna, iluminação indireta, modo automático por luminosidade, modo respiração e controle manual de brilho.

## Justificativa

A LumenBABY será o primeiro produto real da marca LumenSense e servirá como estudo de caso prático para o futuro curso Projetista 360°.

O projeto existe para validar, na prática, o processo de criação de um produto eletrônico autoral, desde a ideia até a possível comercialização controlada.

## Filosofia do projeto

O projeto prioriza:

- simplicidade;
- replicabilidade;
- custo acessível;
- experiência visual;
- autonomia suficiente;
- facilidade de ensino;
- potencial comercial.

## Escopo inicial

Inclui:

- firmware Arduino;
- controle de brilho por PWM;
- leitura de LDR;
- botões físicos;
- modo automático;
- modo respiração;
- modo manual;
- documentação técnica;
- preparação para protótipo;
- versionamento público no GitHub.

Não inclui:

- Wi-Fi;
- Bluetooth;
- aplicativo;
- PCB própria;
- ESP32;
- Raspberry Pi Pico;
- controle remoto;
- display;
- monitoramento de bebê;
- produto industrial em larga escala.

## Hardware previsto

- Arduino Pro Mini 5V / 16 MHz;
- LDR;
- 2 botões;
- IRLZ44N;
- fita LED 3528 de 12 V;
- MT3608;
- TP4056 com proteção;
- 2 baterias 18650 em paralelo;
- chave liga/desliga;
- capacitores;
- resistores;
- MDF 3 mm.

## Critério de sucesso

O projeto será considerado bem-sucedido nesta fase se o firmware permitir validar os modos principais da luminária de forma estável, simples e didática, com documentação pública coerente e histórico versionado.
