# Decisão 0001 — Repositório público desde o início

## Contexto

O Projeto LumenBABY será desenvolvido no VSCode, com apoio do Codex como IA geradora de código, e terá firmware totalmente em Arduino.

Além disso, o projeto será versionado desde o início e publicado no GitHub em repositório público.

## Decisão

O projeto deve ser estruturado considerando visibilidade pública desde a primeira etapa.

## Justificativa

Como a LumenBABY também servirá como estudo de caso real para o futuro curso Projetista 360°, o histórico público de desenvolvimento poderá demonstrar processo, método, decisões técnicas e evolução do produto.

## Consequências

A partir desta decisão:

- nenhum arquivo sensível deve ser versionado;
- documentos privados devem ficar fora do repositório;
- custos internos, dados de clientes e estratégias comerciais privadas não devem entrar no GitHub;
- toda alteração relevante deve manter a documentação coerente;
- commits devem ser claros e rastreáveis.
