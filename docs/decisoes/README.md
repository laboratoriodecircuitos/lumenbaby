# Decisões do Projeto

Esta pasta deve registrar decisões importantes do Projeto LumenBABY.

Cada decisão relevante deve explicar:

- data;
- contexto;
- decisão tomada;
- justificativa;
- alternativas consideradas;
- impacto no projeto.

Exemplos de decisões que devem ser registradas:

- escolha do Arduino Pro Mini 5V;
- uso de TP4056;
- uso de MT3608;
- escolha da fita LED 12 V;
- definição dos modos de operação;
- alterações relevantes no hardware ou firmware.
