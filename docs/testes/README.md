# Testes do Projeto LumenBABY

Esta pasta deve concentrar registros de testes do projeto.

Testes previstos:

- teste de compilação no Arduino IDE;
- teste de leitura do LDR;
- teste de PWM da fita LED;
- teste de fade in e fade out;
- teste do modo respiração;
- teste dos botões;
- teste de autonomia;
- teste de aquecimento;
- teste de recarga;
- teste de montagem física;
- teste com usuário real.

## Registros de teste

- [Teste 0001 — Compilação no Arduino IDE](0001_compilacao_arduino_ide.md)

Cada teste deve registrar:

- data;
- objetivo;
- montagem usada;
- resultado esperado;
- resultado obtido;
- problemas encontrados;
- decisão após o teste.

Não registrar dados pessoais de usuários testadores no repositório público.
